# 批发商城开工包-菜单页面接口映射矩阵

## 1. 说明

本文档用于将菜单、页面、接口三层结构固定下来，作为后续任务拆分、联调和 Swagger 补齐依据。

- 菜单：管理端或业务侧导航入口
- 页面：原型图中的独立交互页
- 接口：页面最少需要的主列表、详情、动作接口

## 6. 页面字段覆盖要求

每个页面至少满足以下要求：

- 列表接口覆盖筛选字段、分页字段、状态字段、排序字段
- 详情接口覆盖页面所有展示字段
- 动作接口覆盖用户实际可点击操作
- 所有状态按钮必须能映射到独立动作接口，不允许前端本地伪流转

## 7. 联调要求

- Swagger 输出时按本矩阵归档 Tag
- 管理端、供货端、代理端、消费者端分别展示
- 若一个页面缺少“列表/详情/动作”三类接口中的任一项，开发前必须先补文档再编码
