# 菜单维护功能验收清单

更新时间：2026-06-26

## 说明

- `已完成`：当前前端页面已经对齐本轮约定能力，且已接入真实后端或明确的只读能力。
- `只读完成`：该菜单按当前后端能力只承接列表、详情、状态展示、导出等只读动作，不强行补新增/删除。
- `配置完成`：该菜单属于配置型页面，不以列表 CRUD 为主，已按配置页标准补齐状态展示、图片上传、缺项提示等。
- `隐藏路由完成`：当前侧边栏不一定直显，但路由和页面能力已补齐。

## 总体结论

- 正式业务主菜单已基本完成本轮矩阵要求。
- 核心缺口已补齐到商品、营销、订单、采购、资金、会员、消息、内容、支持、权限、系统日志。
- 当前更接近“验收收口”阶段，不再是大规模功能开发阶段。
