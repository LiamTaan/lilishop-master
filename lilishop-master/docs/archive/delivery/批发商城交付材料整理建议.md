# 批发商城交付材料整理建议

## 1. 对客户发送建议

- 优先发送汇报版、一页总结、总说明、验收清单、FAQ
- 不建议首次就把全部阶段文档一次性发给客户
- 联调类文档按需要再补发给前端或测试

## 2. Word/PDF 建议顺序

建议整理成一个总包文档时，顺序如下：

1. 批发商城最终发客户版总包说明
2. 批发商城客户汇报一页总结
3. 批发商城后端客户交付总说明
4. 批发商城客户验收总清单
5. 批发商城客户常见问题清单

## 3. 附件建议

- 接口总对照表作为附件 1
- 字段映射说明作为附件 2
- 状态流转说明作为附件 3

## 4. 汇报口径建议

- 先讲“已交付哪些业务域”
- 再讲“哪些三方复用，哪些业务补齐”
- 最后讲“当前已可联调、可验收，剩余是页面接入和细节收口”
