# 批发商城阶段三样例请求清单

## 1. 说明

本清单用于阶段三后端联调与冒烟验证。

## 2. 店铺端样例

### 2.1 订单发货

```http
PUT /store/order/order/order-sn-001/deliver
Content-Type: application/x-www-form-urlencoded

logisticsNo=SF123456789
logisticsId=logistics-id-001
```

### 2.2 订单发货旧接口

```http
POST /store/order/order/order-sn-001/delivery
Content-Type: application/x-www-form-urlencoded

logisticsNo=SF123456789
logisticsId=logistics-id-001
```

### 2.3 查询店铺核销记录

```http
GET /store/order/verificationRecord?pageNumber=1&pageSize=10
```

## 3. 买家端样例

### 3.1 核销记录查询

```http
GET /buyer/order/verificationRecord?pageNumber=1&pageSize=10
```

## 4. 管理端样例

### 4.1 查询核销记录

```http
GET /manager/other/verificationRecord?pageNumber=1&pageSize=10
```

## 5. 核销样例

### 5.1 按核销码查询订单

```http
GET /store/order/order/getOrderByVerificationCode/verification-code-001
```

### 5.2 核销订单

```http
PUT /store/order/order/order-sn-001/verification-code-001
```
