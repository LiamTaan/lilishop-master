# 批发商城阶段五样例请求清单

## 1. 说明

本清单用于阶段五第一轮后端联调与冒烟验证。

## 2. 买家端样例

### 2.1 查询钱包余额

```http
GET /buyer/wallet/wallet
```

### 2.2 查询提现设置

```http
GET /buyer/wallet/wallet/withdrawalSettingVO
```

### 2.3 提交提现申请

```http
POST /buyer/wallet/wallet/withdrawal
Content-Type: application/x-www-form-urlencoded

price=100&realName=ZhangSan&connectNumber=wechat-openid-001
```

### 2.4 查询提现记录

```http
GET /buyer/member/withdrawApply?pageNumber=1&pageSize=10
```

### 2.5 查询钱包流水

```http
GET /buyer/wallet/log?pageNumber=1&pageSize=10
```

## 3. 平台端样例

### 3.1 查询会员钱包

```http
GET /manager/wallet/wallet?memberId=member-id-001
```

### 3.2 运营手工加款

```http
PUT /manager/wallet/wallet/increase?memberId=member-id-001&rechargeMoney=50
```

### 3.3 查询提现申请

```http
GET /manager/wallet/withdrawApply?pageNumber=1&pageSize=10&applyStatus=APPLY
```

### 3.4 审核提现申请

```http
POST /manager/wallet/withdrawApply
Content-Type: application/x-www-form-urlencoded

applyId=withdraw-id-001&result=true&remark=审核通过
```

### 3.5 查询充值记录

```http
GET /manager/wallet/recharge?pageNumber=1&pageSize=10
```

## 4. 店铺端样例

### 4.1 查询结算单分页

```http
GET /store/order/bill/getByPage?pageNumber=1&pageSize=10&billStatus=OUT
```

### 4.2 查询结算单详情

```http
GET /store/order/bill/get/bill-id-001
```

### 4.3 查询结算单流水

```http
GET /store/order/bill/bill-id-001/getStoreFlow?pageNumber=1&pageSize=10&flowType=PAY
```

### 4.4 核对结算单

```http
PUT /store/order/bill/check/bill-id-001
```

## 5. 代理商入口回归样例

### 5.1 查询代理商角色

```http
GET /buyer/agent/role/check
```

### 5.2 查询代理商已绑定店铺

```http
GET /buyer/agent/store
```
