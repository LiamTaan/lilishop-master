# 批发商城阶段四样例请求清单

## 1. 说明

本清单用于阶段四后端联调与冒烟验证。

## 2. 买家端样例

### 2.1 拼团商品列表

```http
GET /buyer/promotion/pintuan?pageNumber=1&pageSize=10&goodsName=苹果&categoryPath=0,100,200
```

### 2.2 拼团活动详情

```http
GET /buyer/promotion/pintuan/pintuan-id-001
```

### 2.3 待成团成员

```http
GET /buyer/promotion/pintuan/pintuan-id-001/members
```

### 2.4 拼团分享信息

```http
GET /buyer/promotion/pintuan/share?parentOrderSn=parent-order-sn-001&skuId=sku-id-001
```

## 3. 店铺端样例

### 3.1 查询拼团活动分页

```http
GET /store/promotion/pintuan?pageNumber=1&pageSize=10&promotionName=夏季拼团
```

### 3.2 查询拼团活动详情

```http
GET /store/promotion/pintuan/pintuan-id-001
```

### 3.3 查询拼团活动商品

```http
GET /store/promotion/pintuan/goods/pintuan-id-001?pageNumber=1&pageSize=10
```

### 3.4 新增拼团活动

```http
POST /store/promotion/pintuan
Content-Type: application/json

{
  "promotionName": "批发夏日拼团",
  "startTime": "2026-06-18 00:00:00",
  "endTime": "2026-06-25 23:59:59",
  "requiredNum": 3,
  "limitNum": 2,
  "promotionGoodsList": [
    {
      "skuId": "sku-id-001"
    }
  ]
}
```

### 3.5 调整拼团活动状态

```http
PUT /store/promotion/pintuan/status/pintuan-id-001?startTime=1781712000000&endTime=1782316799000
```

## 4. 平台端样例

### 4.1 查询拼团活动分页

```http
GET /manager/promotion/pintuan?pageNumber=1&pageSize=10&promotionStatus=START
```

### 4.2 查询拼团活动商品

```http
GET /manager/promotion/pintuan/goods/pintuan-id-001?pageNumber=1&pageSize=10
```

### 4.3 查询待成团成员

```http
GET /manager/promotion/pintuan/pintuan-id-001/members
```

### 4.4 批量调整拼团状态

```http
PUT /manager/promotion/pintuan/status/pintuan-id-001,pintuan-id-002?startTime=1781712000000&endTime=1782316799000
```

## 5. 营销回归样例

### 5.1 买家端优惠券列表

```http
GET /buyer/promotion/coupon?pageNumber=1&pageSize=10
```

### 5.2 店铺端满减列表

```http
GET /store/promotion/fullDiscount?pageNumber=1&pageSize=10
```
