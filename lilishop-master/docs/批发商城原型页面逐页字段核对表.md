# 批发商城原型页面逐页字段核对表

历史需求分析文档已不再保留在仓库内。当前逐页核对时，以接口返回、状态流转和测试主文档为准。

推荐搭配使用：

- `批发商城后端全业务测试用例总文档.md`
- `批发商城管理端接口总对照表.md`
- `批发商城后端状态流转与治理动作说明.md`

| 原型页面 | 核对重点 |
|---|---|
| 角色选择 | 角色编码、角色描述、角色切换 |
| 登录注册 | 手机号、验证码、协议勾选 |
| 供货商店铺申请 | 主体类型、资质、法人、审核状态 |
| 首页/分类 | banner、shortcutNav、分类、商品卡片 |
| 商品详情 | 规格、价格、批发阶梯价、库存、起购量 |
| 购物车 | 商品项、勾选、金额、备注、地址 |
| 地址管理 | 收货人、手机号、区域、默认地址 |
| 订单列表/详情 | 订单头、订单项、状态、核销码 |
| 售后申请 | 申请类型、原因、金额、凭证 |
| 拼团详情 | 拼团价、人数、截止时间、状态 |
| 核销详情 | 核销码、核销状态、核销人、核销时间 |
| 优惠券 | 券类型、门槛、状态、有效期 |
| 积分商城 | 积分值、库存、兑换状态 |
| 钱包 | 余额、可提现、冻结、待结算 |
| 代理商我的 | 钱包、券数、收藏、浏览记录 |
| 消费者我的 | 钱包、积分、券数、收藏 |
| 供货商我的 | 店铺、收益、商品数、待结算 |
| 供货商工作台 | 营业额、订单量、在售商品、待发货 |
| 供货商数据概览 | 销售额、退款额、排行、趋势 |
| 供货商卡券管理 | 卡券类型、面值、库存、状态 |
| 供货商资产管理 | 累计收益、已结算、可提现、冻结 |
| 代理商经营概览 | 销售额、订单数、绑定店铺、佣金 |
| 代理商资金与采购对账 | 余额、已结算、未结算、对账状态 |
| 代理商批发页 | 批发价、起批量、销量、库存 |
| 我的店铺 | 绑定状态、区域、分类、联系人 |
| 平台工作台/数据概览 | 总收益、活跃门店、排行、拼团率 |
