# 批发商城客户验收总清单

## 1. 文档目的

本文档用于客户阶段验收、内部演示验收和测试总回归。

验收原则：

- 以当前仓库已落地后端能力为准。
- 以原型业务域为验收分组，不按代码模块分散验收。
- 每个业务域同时关注列表、汇总、详情、状态动作、历史记录五类能力。

## 2. 平台工作台验收

- 可访问 `GET /manager/dashboard/wholesale`
- 可返回首页基础概览
- 可返回热卖店铺排行
- 可返回热卖商品排行
- 可返回类目销售排行
- 可返回平台分账余额概览
- 可返回代理商采购对账汇总
- 可返回代理商资金对账汇总
- 可返回店铺治理汇总
- 可返回核销治理汇总
- 可返回平台治理总览

## 3. 店铺治理验收

- 可分页查询店铺列表
- 可查看店铺治理汇总
- 可查看店铺详情
- 可查看店铺审核历史
- 可执行店铺审核通过
- 可执行店铺审核驳回
- 可执行店铺冻结
- 可执行店铺启用
- 审核状态流转符合既定口径

## 4. 代理商治理验收

- 可创建代理商身份
- 可查询代理商身份列表
- 可创建代理商店铺绑定
- 可分页查询代理商店铺绑定
- 可审核代理商绑定通过
- 可审核代理商绑定驳回
- 可解绑代理商绑定
- 绑定状态流转符合既定口径

## 5. 订单治理验收

- 可分页查询订单
- 可查看订单数量统计
- 可查看订单治理汇总
- 可查看订单详情
- 可执行平台确认收款
- 可执行平台取消订单
- 可写入平台订单备注
- 订单状态流转符合既定口径

## 6. 售后治理验收

- 可分页查询售后单
- 可查看售后数量统计
- 可查看售后治理汇总
- 可查看售后详情
- 可查询退货物流轨迹
- 可查询售后收件地址
- 可执行平台审核售后
- 可执行平台退款
- 售后状态流转符合既定口径

## 7. 核销治理验收

- 可分页查询核销记录
- 可查看核销治理汇总
- 可分页查询异常核销
- 可查看异常核销治理汇总
- 可查询异常核销导出列表
- 可按成功、失败、异常口径区分统计
- 可按买家端、店铺端、平台端来源区分统计

## 8. 分账与对账验收

- 可查询分账规则列表
- 可查询分账记录分页
- 可查询分账记录导出列表
- 可查看分账余额概览
- 可查看分账治理汇总
- 可查看代理商对账总览
- 可分页查询代理商采购对账
- 可查看代理商采购对账汇总
- 可查看代理商采购对账详情
- 可分页查询代理商资金对账
- 可查看代理商资金对账汇总
- 可查看代理商资金对账详情

## 9. 编译与联调基线验收

- 使用 `D:\utils\jdk21` 可成功编译 `framework`、`manager-api`、`buyer-api`、`seller-api`、`common-api`、`im-api`
- Swagger 可浏览管理端、买家端、商家端、公共服务、IM 的接口文档
- 当前顶层 `.http` 联调请求可覆盖管理端关键链路，阶段性请求已归档到 `docs/archive/stages/`
- 交付文档、状态流转说明、接口清单均已存在
- 无 app 界面时，可通过 `buyer-api + common-api + seller-api` 接口模拟 app 与商家侧主链路

## 10. 建议验收顺序

1. 先验平台工作台和店铺治理
2. 再验代理商治理和订单治理
3. 再验售后治理和核销治理
4. 最后验分账与对账治理
